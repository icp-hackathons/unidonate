## Hackathon submission

Entry for the 2024 South Africa Blockchain in Education Hackathon

Link: https://summit.immersiveeducation.org/SouthAfrica/2024/hackathon-start.html

This example represents a decentralized system designed for public donations directed towards students and schools via Bitcoin. The solution is implemented on the Internet Computer platform and internally leverages [Bitcoin integration](https://wiki.internetcomputer.org/wiki/Bitcoin_Integration) of the Internet Computer.
